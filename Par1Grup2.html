Fork Join Pool

El ForkJoinPoolse agregó a Java en Java 7. El ForkJoinPooles similar al Java ExecutorService pero con una diferencia. El ForkJoinPool facilita que las tareas dividan su trabajo en tareas más pequeñas que luego se envían al ForkJoinPool. Las tareas pueden seguir dividiendo su trabajo en subtareas más pequeñas durante el tiempo que sea necesario para dividir la tarea.
